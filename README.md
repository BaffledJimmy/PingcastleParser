## Getting Up and Running

* Read Blogpost
* Clone Repo
* Place files in C:\Scripts
* Create any Webhook URls required (Slack / Teams)
* Create Scheduled Task to run PingCastle, and one to upload your HTML reports to OpenAI or your LLM of choice.
